ChattisVani

Empowering Chhattisgarh with Local Language AI

ChattisVani is a revolutionary application designed to bridge the digital divide and bring the power of Large Language Models (LLMs) to the tribal and local communities of Chhattisgarh. Recognizing the importance of communication in one's native tongue, this application enables users to interact with advanced AI technologies using their local languages.

Our mission is to make AI accessible and useful for everyone in Chhattisgarh, fostering digital inclusion and empowering communities by providing information and services in a language they understand and are comfortable with.

Features

  • Local Language Support: Interact with the AI using various local languages spoken in Chhattisgarh.
  • LLM Integration: Leverages the capabilities of advanced Large Language Models to provide intelligent responses and assistance.
  • User-Friendly Interface: Designed with simplicity in mind to ensure ease of use for all members of the community.

AI Model

This application leverages the power of a Large Language Model (LLM) to understand and generate responses in local languages.

Getting Started

To get your local copy up and running, follow these simple steps.

Prerequisites

Make sure you have Node.js and npm (or yarn/pnpm) installed on your system.

Installation

  1. Clone the repository: 
    git clone https://github.com/sameer-banchhor-git/ChattisVani
  2. Navigate to the project directory: 
    cd ChattisVani
  3. Install dependencies: 
    npm install
# or yarn install
# or pnpm install

Running the Application

To run the application in development mode:

npm run dev
# or yarn dev
# or pnpm dev

Open http://localhost:3000 with your browser to see the result.
