Method of False Position

Find a real root of: x³ - 2x - 5 = 0

What is the Method of False Position?

The Method of False Position (Regula Falsi) is a numerical method for finding roots of equations. It combines the reliability of the bisection method with faster convergence by using linear interpolation instead of simple bisection.

Step 1: Define the Function and Find Initial Interval

1 Function: f(x) = x³ - 2x - 5
We need to find an interval [a, b] where f(a) and f(b) have opposite signs, ensuring a root exists between them.
2 Testing values:
f(2) = 2³ - 2(2) - 5 = 8 - 4 - 5 = -1 < 0
f(3) = 3³ - 2(3) - 5 = 27 - 6 - 5 = 16 > 0
Since f(2) < 0 and f(3) > 0, there's a root between x = 2 and x = 3.

Step 2: Apply the False Position Formula

3 Formula:
x₁ = a - f(a) × (b - a) / (f(b) - f(a))
This formula finds the x-intercept of the line connecting points (a, f(a)) and (b, f(b)).

Step 4: Detailed Calculation Example

4 First Iteration Calculation:
a = 2, b = 3
f(a) = f(2) = -1
f(b) = f(3) = 16

x₁ = a - f(a) × (b - a) / (f(b) - f(a))
x₁ = 2 - (-1) × (3 - 2) / (16 - (-1))
x₁ = 2 - (-1) × 1 / 17
x₁ = 2 + 1/17
x₁ = 2 + 0.0588
x₁ = 2.0588
