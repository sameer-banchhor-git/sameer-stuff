üéØ Root Finding Methods

1. Bisection Method

Basic Formula:
$$c = \frac{a + b}{2}$$

Where $a$ and $b$ are endpoints of interval such that $f(a) \cdot f(b) < 0$

Iteration Formula:
$$x_{n+1} = \frac{x_n + x_{n-1}}{2}$$
Error Bound:
$$|x_n - r| \leq \frac{b - a}{2^n}$$

Where $r$ is the actual root and $n$ is the number of iterations

Number of Iterations Required:
$$n = \frac{\log(b - a) - \log(\epsilon)}{\log(2)}$$

Where $\epsilon$ is the desired accuracy

Algorithm Steps:
1. Check: $f(a) \cdot f(b) < 0$
2. $c = \frac{a + b}{2}$
3. If $f(a) \cdot f(c) < 0$, then $b = c$; else $a = c$
4. Repeat until convergence

2. Regula Falsi (False Position) Method

Main Formula:
$$x_{n+1} = x_0 - f(x_0) \cdot \frac{x_1 - x_0}{f(x_1) - f(x_0)}$$
Alternative Form:
$$c = \frac{a \cdot f(b) - b \cdot f(a)}{f(b) - f(a)}$$

Where $a$ and $b$ are endpoints with $f(a) \cdot f(b) < 0$

General Iteration:
$$x_n = \frac{x_{n-2} \cdot f(x_{n-1}) - x_{n-1} \cdot f(x_{n-2})}{f(x_{n-1}) - f(x_{n-2})}$$
Convergence: Linear convergence, faster than bisection but slower than Newton's method

3. Newton-Raphson Method

Main Formula:
$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$
Error Formula:
$$e_{n+1} = -\frac{f''(\xi)}{2f'(x_n)} \cdot e_n^2$$

Where $e_n = x_n - r$ (error at $n$-th iteration)

Convergence Condition:
$$|f(x_0) \cdot f''(x_0)| < [f'(x_0)]^2$$
Modified Newton's Method (for multiple roots):
$$x_{n+1} = x_n - m \cdot \frac{f(x_n)}{f'(x_n)}$$

Where $m$ is the multiplicity of the root

Advantages: Quadratic convergence when conditions are met
Disadvantages: Requires derivative calculation, may not converge if initial guess is poor

üìê Solution of Cubic Equations

General Cubic Equation

Standard Form:
$$ax^3 + bx^2 + cx + d = 0, \quad (a \neq 0)$$
Reduced Form (Depressed Cubic):
$$t^3 + pt + q = 0$$

Obtained by substitution $x = t - \frac{b}{3a}$

Where: $p = \frac{3ac - b^2}{3a^2}$ and $q = \frac{2b^3 - 9abc + 27a^2d}{27a^3}$

Cardano's Method

Discriminant:
$$\Delta = -4p^3 - 27q^2$$
Case 1: $\Delta > 0$ (Three real roots)
$$t_k = 2\sqrt{-\frac{p}{3}} \cos\left(\frac{1}{3}\arccos\left(\frac{3q}{2p}\sqrt{-\frac{3}{p}}\right) - \frac{2\pi k}{3}\right)$$

For $k = 0, 1, 2$

Case 2: $\Delta < 0$ (One real root)
$$t = \sqrt[3]{-\frac{q}{2} + \sqrt{\frac{q^2}{4} + \frac{p^3}{27}}} + \sqrt[3]{-\frac{q}{2} - \sqrt{\frac{q^2}{4} + \frac{p^3}{27}}}$$
Case 3: $\Delta = 0$ (Multiple roots)

If $p = q = 0$: Triple root $t = 0$

If $p \neq 0$: $t_1 = \frac{3q}{p}$, $t_2 = t_3 = -\frac{3q}{2p}$

üî¢ Solution of Biquadratic (Quartic) Equations

General Quartic:
$$ax^4 + bx^3 + cx^2 + dx + e = 0$$
Depressed Quartic:
$$x^4 + px^2 + qx + r = 0$$

Obtained by substitution $x = y - \frac{b}{4a}$

Ferrari's Method

Resolvent Cubic:
$$t^3 - pt^2 - 4rt + 4pr - q^2 = 0$$

Let $t_0$ be any root of this cubic

Quartic Solutions:
$$x = \pm\frac{1}{2}\sqrt{t_0} \pm \frac{1}{2}\sqrt{-t_0 - p \pm \frac{2q}{\sqrt{t_0}}}$$

Special Case: Biquadratic (No odd powers)

Form:
$$ax^4 + bx^2 + c = 0$$
Solution:

Let $y = x^2$, then solve $ay^2 + by + c = 0$

$$y = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$ $$x = \pm\sqrt{y}$$

üåü Complex Roots of Polynomial Equations

Fundamental Theorem of Algebra

Every polynomial of degree $n$ has exactly $n$ roots (counting multiplicities) in the complex plane.

Complex Root Theorems

Complex Conjugate Root Theorem:

If $a + bi$ is a root of a polynomial with real coefficients, then $a - bi$ is also a root.

Quadratic Formula (Complex Roots)

For $ax^2 + bx + c = 0$:
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

When $\Delta = b^2 - 4ac < 0$:

$$x = \frac{-b \pm i\sqrt{4ac - b^2}}{2a}$$

De Moivre's Theorem

For complex number $z = r(\cos\theta + i\sin\theta)$:
$$z^n = r^n(\cos(n\theta) + i\sin(n\theta))$$
$n$-th Roots of Unity:
$$\omega_k = e^{2\pi i k/n} = \cos\left(\frac{2\pi k}{n}\right) + i\sin\left(\frac{2\pi k}{n}\right)$$

For $k = 0, 1, 2, ..., n-1$

Synthetic Division for Complex Roots

If $\alpha$ is a root of $P(x)$:
$$P(x) = (x - \alpha) \cdot Q(x)$$

Where $Q(x)$ is obtained by synthetic division

Newton's Method for Complex Roots

Complex Newton-Raphson:
$$z_{n+1} = z_n - \frac{f(z_n)}{f'(z_n)}$$

Works for complex starting points $z_0$
